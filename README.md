# hubot-wmata

Get status information from WMATA

See [`src/wmata.coffee`](src/wmata.coffee) for full documentation.

## Installation

In hubot project repo, run:

`npm install hubot-wmata --save`

Then add **hubot-wmata** to your `external-scripts.json`:

```json
[
  "hubot-wmata"
]
```

## Sample Interaction

```
user1>> hubot hello
hubot>> hello!
```
